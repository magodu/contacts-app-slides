Details and enhancements

Now the application it's almost finished, but we can improve some things:

  • Add Lodash library: A very useful JS library.
  • Default value: In ContactComponent we can add a default value in case that any value was empty.
  • OnDestroy Lifehook: Another life hook ( like OnInit) that can be used to remove a suscription when the component is not used, and prevent memory leaks.
  • getContact function refactor: We was using the id as index, but this is not a good practice if ids changes.
