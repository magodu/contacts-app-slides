Angular 2 Forms

Forms are one the most crucial aspect in a web application. Angular 2 has a good forms support.

There are two ways to build forms in Angular 2:

  • Template-driven forms: the form logic is in the template. It's recommended for simple forms.
  • Model-driven forms: all form logic is in the component. They are more powerful, keeps the template clean and allows more complex logic and validations. All the form have to be built programatically.

We'll see both, starting with the template-driven way.
