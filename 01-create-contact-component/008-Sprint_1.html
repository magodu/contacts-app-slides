Sprint 1 - Create a Product component

We propose to you build an Angular 2 Shop application. First part will be create a product.

  1. Create workspace with angular-CLI (or another starter)
  2. Identify app.module, main.ts and app.component
  3. Display one product in app.component (without creating them in a separate component)
  4. Display product tags
  5. Display product using separate component product.component
  6. Create Product class
  7. Use currency pipe for prices
  8. Component design: do your app prettier
